# Algorithm & Spring Boot Interview Practice

A hands-on study repo for Java, Spring Boot, and algorithms — built to pass technical interviews.

## Structure

```
├── algorithms/       # Data structures & algorithm problems
├── java-core/        # Core Java concepts (OOP, collections, concurrency, etc.)
└── spring-boot/      # Spring Boot concepts and mini-projects
```

## Topics Covered

### Algorithms & Data Structures
- Arrays, Strings, Linked Lists
- Trees, Graphs, Heaps
- Sorting & Searching
- Dynamic Programming
- Recursion & Backtracking

### Java Core
- OOP principles
- Collections framework
- Generics, Lambdas, Streams
- Multithreading & Concurrency
- Exception handling

### Spring Boot
- REST APIs
- Dependency Injection / IoC
- JPA & Hibernate
- Spring Security basics
- Testing with JUnit & Mockito
